Benefits of using semantic tags in web page

Semantic tags are HTML tags that give meaning to the content of a web page beyond just its presentation. They provide information about the structure and the meaning of the content, making it easier for search engines,screen readers and other tools to understand the page and its content. Unlike traditional HTML tags, semantic tags focus on the meaning of the content, rather than on the visual appearance.

For example:- the < header > tag is a semantic tag indicating that the header section of a page begins, while the < article > tag indicates that the enclosed content is an independent piece of content that can stand alone and so on.
