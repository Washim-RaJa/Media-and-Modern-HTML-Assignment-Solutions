Web Accessibility

Web accessibility refers to the practice of designing and developing web applications that can be accessed and used by people with disabilities or different needs, without barriers or limitations.

List of some of the assistive devices which play a major role in providing accessibility

  1. Screen Reader

    A software that reads out loud the content of a web page.

  2. Voice recognition software

    A software that helps users to navigate web pages & input text using voice commands.

  3. Keyboard alternatives

    Keyboard alternative devices like sip-and-puff, head-tracking and eye-tracking which help people with disabilities to navigate and interact with web pages.
